三盲彳亍+M2法

发表于 2017-05-29 | 分类于 cube

编码法

棱块M2

第一个循环+完整小循环，

  1. 看Df位置上的色块的归位的正确位置所对应的编码，即为第一个编码。
    如果缓冲块正好在位子上但是方向不对，则第一个编码应该找另一个块，位置是Ufl颜色所在位置的对应位置
  2. 第一个编码的位置上的色块的正确位置即为第二个编码，以此类推。
  3. 当编到归位到Df位置上的色块后，第一个小循环结束，如果该色块是正确归位，则寻找下一个需要编码的块的高级面开始编，如果该色块是反向归位，则寻找下一个需要编码的块的低级面开始编，开始进入小循环，小循环必须是个闭环。直至所有需要编码的块都编到。
  4. 如果编码总数是奇数，则需要做i=u2 M’ u2 M将棱块簇的最后形态变Uf和Ub平移交换。
  5. 如果编码中第偶数个编码是ABMN，则需要编为MNAB。

A: U2 MR U2 R
B: D (MR U R2 U’ MR’ U R2 U’) D’ M2
M: MR’ U2 MR’ U2
N: M2 F (R U R’ E’ R U’ R’ E) F’
F: U R’ U’ x’(U’ R2 U M2 U’ R2 U) x U R U’

AM = MA = BN = NB =1
AN = BM

角块彳亍

转换机原理，(1-2-3)的循环可以分解为1-2，3-2,2-1,2-3，其中1-2的操作所影响的块与3-2的操作所影响的块不可以有另外的交集。
编码法:
缓冲块Ufl

  1. 第一编码是Ufl所在的位置。依次。。直到编到Ufl块(不编)第一个循环结束。
  2. 第二个循环的起始码是 前码的颜色所属的对应位置(不是所在的对应位置)。
  3. 第三个循环的起始码是 前码的颜色所在的对应位置。
  4. 上述的理论同样适用于单个块翻色的情况，但操作时应当插入jj作为过渡。

实例1（s=Y/R i=I）

打乱

U F R2 L B L B2 R’ F U F2 U2 F2 B2 D’ R2 U2 F2 U L2

编码

EF IN LG OX SO
GD CE LW NN OY SE 注意EF应当解析为EjjF
